Servidores de Aplicação

Este site é construído com WildFly e MaterializeCSS para a aula de Servidores de Aplicação

O que é um projeto Web Java e como isso se relaciona com IDEs Eclipse e Java?

Um projeto Web Java é um contexto de desenvolvimento de uma aplicação Web em Java e pode ser desenvolvida através de IDEs, Integrated Development Enviroment, ou Ambiente Integrado de Desenvolvimento, como o Eclipse ou Netbeans, dentre outras. A IDE facilita o trabalho do desenvolvedor oferecendo recursos automáticos e plug-ins gerenciais para o projeto.

O que é um arquivo .WAR?

War vem de Web Application Resource, ou Recurso de Aplicação Web. É um arquivo JAR utilizado para distribuir uma coleção de páginas JavaServer, Servlets, Classes Java, XML e páginas estáticas na web.

O que é e como implantar WAR no WildFly?

O WildFly é um servidor de aplicação Java EE também conhecido como JBoss, desenvolvido em Java e compatível com qualquer sistema operacional com suporte ao Java.

Para implementar um arquivo WAR no WildFly basta seguir até a página de administração, clicar em Deployments, clicar em Add, selecionar o arquivo WAR para ser implementado e voilà, a sua aplicação web está pronta para ser usada e distribuída.
